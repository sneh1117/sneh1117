### [StitchTales](https://github.com/sneh1117/stitchtales) — Blogging Platform for the Maker Community
> **Live in production** | **Full-featured CMS** | **Zero-dependency paywall**

<img width="1272" height="611" alt="StitchTales Home" src="https://github.com/user-attachments/assets/ef07cc79-54cf-4ba5-838c-1b9f17e8ee37" />

A Django-powered blogging platform built for craft creators — featuring rich text editing, visitor analytics, and a unique 3-second preview paywall to drive user registrations.

**Key Technical Wins:**
- 🔒 **Soft paywall system** — 3-second preview → blur → signup modal (vanilla JS, no frameworks)
- 📊 **Built-in analytics** — no Google Analytics, full data ownership, GDPR-friendly
- 🖼️ **Custom Supabase Storage backend** — replaced AWS S3 with simpler REST API
- ⚡ **HTMX interactions** — like/bookmark buttons update without page reload

**Tech Stack:** Django • PostgreSQL • Supabase Storage • HTMX • Quill.js • Railway

**What Makes It Different:**
- All analytics tracked in-house (visitor geo-location, referrers, top pages)
- View count integrity (excludes author's own views)
- Multiple cover images per post with Bootstrap carousel
